# PDFmakerWithExcel
Se crean PDF a partir de la tabla de excel, puedes añadir nuevos alumnos o cambiarlo a tu gusto

---

# Pasos necesarios para que funcione

1. Descarga el intérprete de Python 3.11 o superior
2. Descarga las librerías necesarias con "pip install requirements.txt"
3. Puedes añadir o modificar según tus necesidades el código y modificar las coordenadas de donde se genere
